    Создание элементов

    Создавайте новые элементы, комбинируя существующие друг с другом.

    Удаление элементов

    Чтобы удалить элемент, захватите его и перенесите в корзину в нижней части экрана.

    Добавление элементов

    Чтобы добавить уже созданный элемент на рабочую область, выберите его из каталога доступных элементов и нажмите кнопку «Добавить».

    Копирование элементов

    Чтобы скопировать элемент, кликните по нему два раза.

    Символ * около элемента означает, что данный элемент конечный.
